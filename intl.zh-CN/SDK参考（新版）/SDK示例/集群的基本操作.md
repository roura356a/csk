# 集群的基本操作

本文介绍ACK集群的基本操作示例，包括创建集群、查集群状态、扩容集群、查集群节点和删除集群。

## 安装Tea-console

调试SDK示例前需要安装Tea-console，安装方式如下：

-   Java SDK

通过Maven进行安装，请在项目文件中加入以下依赖：

```
	<dependency>
  		<groupId>com.aliyun</groupId>
  		<artifactId>tea-console</artifactId>
  		<version>0.0.1</version>
	</dependency>
```

-   Python SDK

推荐使用PyPI进行安装，执行以下命令：

```
pip install alibabacloud_tea_console
```

-   GO SDK

推荐使用go工具进行安装，执行以下命令：

```
go get github.com/alibabacloud-go/tea-console/client:v1.0.0
```

-   TypeScript/Node.js SDK

通过NPM工具进行安装，执行以下命令：

```
npm install @alicloud/tea-console@^1.0.0 --save
```
