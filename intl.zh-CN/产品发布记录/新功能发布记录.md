# 新功能发布记录

本文为您介绍了容器服务Kubernetes版（ACK）相关内容的最新动态。

-   容器服务Kubernetes版支持的Kubernetes（K8s）版本：v1.20.4、v1.18.8、v1.16.9。

-   容器服务Kubernetes版支持的操作系统：CentOS 7.x、AliyunLinux 2.1903、Windows Server 2019和Windows Server Core，1909。
