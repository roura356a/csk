# 查询集群可安装组件详情

通过执行命令`arc-cs addon list <cluster_id>`查询集群中可安装的组件详情。

关于该命令的详细参数说明，请参见[查询集群组件版本](/intl.zh-CN/API参考/组件/查询集群组件版本.md)。

请求示例：

```
arc-cs addon list caa73d115830746e0942c9554ecae****
```
